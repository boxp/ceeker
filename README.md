# ceeker

Auto C build command tool.

## Dependencies

- leiningen
- jre

## Generate standalone jar archive

>  lein uberjar

## Usage 

1. Install "ceeker" boot script.

2. Create new ceeker project

> ceeker new hoge

3. Build project.

> ceeker build

4. Start auto build.

> ceeker auto

# ceeker

Auto C build command tool.

## Dependencies

- leiningen
- jre

## Generate standalone jar archive

>  lein uberjar

## Usage 

- Install "ceeker" boot script.

- Create new ceeker project

> ceeker new hoge

- Build project.

> ceeker build

- Start auto build.

> ceeker auto
